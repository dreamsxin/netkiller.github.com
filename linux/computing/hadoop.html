141.1. 单机安装（CentOS 7 + Server JRE 1.8 + Hadoop-2.8.0）

141.1.1. 安装 Java 环境

这里你可以选择使用 OpenJDK 或者 Oracle 的 Server JRE

[root@localhost ~]# yum install -y java-1.8.0-openjdk

JAVA环境变量配置

			
[root@localhost ~]# cat >> /etc/profile.d/java.sh <<EOF
################################################
### Java environment by neo<netkiller@msn.com>
################################################
export JAVA_HOME=/usr/java/default
export PATH=$PATH:$JAVA_HOME/bin
export CLASSPATH=.:$JAVA_HOME/jre/lib:$JAVA_HOME/lib:$JAVA_HOME/lib/tools.jar
export JAVA_OPTS="-Xms2048m -Xmx4096m"
EOF

如果你使用的是Oracle Java 请到Oracle网站下载的Server JRE 并配置 java classpath

			
# wget --no-cookies --header "Cookie: oraclelicense=accept-securebackup-cookie" http://download.oracle.com/otn-pub/java/jdk/8u131-b11/d54c1d3a095b4ff2b6607d096fa80163/server-jre-8u131-linux-x64.tar.gz
如果你的服务器上没有安装 wget 还可以使用 curl 命令下载
# curl -LO -H "Cookie: oraclelicense=accept-securebackup-cookie" http://download.oracle.com/otn-pub/java/jdk/8u131-b11/d54c1d3a095b4ff2b6607d096fa80163/server-jre-8u131-linux-x64.tar.gz?AuthParam=1498632436_9d59c44e2a4f0ca5bd359423ac5b790b

如果你不需要编译，仅仅用于服务器环境，那么无需下载JDK，Server JRE就可以了

141.1.2. 安装 Hadoop

[root@localhost ~]# cd /usr/local/src
[root@localhost src]# wget http://mirrors.tuna.tsinghua.edu.cn/apache/hadoop/common/hadoop-2.8.0/hadoop-2.8.0.tar.gz
[root@localhost src]# tar zxf hadoop-2.8.0.tar.gz
[root@localhost src]# mv hadoop-2.8.0 /srv/
[root@localhost src]# ln -s /srv/apache-hadoop-2.8.0 /srv/apache-hadoop
[root@localhost src]# chown -R hadoop:hadoop /srv/apache-hadoop*
[root@localhost src]# su - hadoop
[hadoop@localhost hadoop]$ cd /srv/apache-hadoop

默认 Hadoop 使用 JAVA_HOME 设置，如果你想使用的JRE来运行Hadoop可以配置

			
[hadoop@localhost hadoop]$ cp /srv/apache-hadoop/etc/hadoop/hadoop-env.sh{,.original}
[hadoop@localhost hadoop]$ sed -i "25s:\${JAVA_HOME}:/srv/java:" hadoop-env.sh

编辑 etc/hadoop/core-site.xml 配置文件，配置HDFS地址

			
[hadoop@localhost hadoop]$ cp /srv/apache-hadoop/etc/hadoop/core-site.xml{,.original}
			
[hadoop@localhost hadoop]$ cat > /srv/apache-hadoop/etc/hadoop/core-site.xml <<EOF
<?xml version="1.0" encoding="UTF-8"?>
<?xml-stylesheet type="text/xsl" href="configuration.xsl"?>
<configuration>
	<property>
		<name>fs.defaultFS</name>
		<value>hdfs://localhost:9000/</value>
	</property>	
</configuration>
EOF

编辑 /srv/apache-hadoop/etc/hadoop/hdfs-site.xml 配置文件，配置HDFS卷目录

			
[hadoop@localhost hadoop]$ cp /srv/apache-hadoop/etc/hadoop/hdfs-site.xml{,.original}

[hadoop@localhost hadoop]$ cat > /srv/apache-hadoop/etc/hadoop/hdfs-site.xml <<EOF
<?xml version="1.0" encoding="UTF-8"?>
<?xml-stylesheet type="text/xsl" href="configuration.xsl"?>
<configuration>
	<property>
		<name>dfs.data.dir</name>
		<value>file:///opt/hadoop/volume/datanode</value>
	</property>
	<property>
		<name>dfs.name.dir</name>
		<value>file:///opt/hadoop/volume/namenode</value>
	</property>
</configuration>
EOF

编辑 /srv/apache-hadoop/etc/hadoop/mapred-site.xml 配置文件

			
[hadoop@localhost hadoop]$ cp /srv/apache-hadoop/etc/hadoop/mapred-site.xml{,.original}

[hadoop@localhost hadoop]$ cat > /srv/apache-hadoop/etc/hadoop/mapred-site.xml <<"EOF"
<?xml version="1.0" encoding="UTF-8"?>
<?xml-stylesheet type="text/xsl" href="configuration.xsl"?>
<configuration>
	<property>
		<name>mapreduce.framework.name</name>
		<value>yarn</value>
	</property>
</configuration>
EOF

编辑 /srv/apache-hadoop/etc/hadoop/yarn-site.xml 配置文件

			
[hadoop@localhost hadoop]$ cp /srv/apache-hadoop/etc/hadoop/yarn-site.xml{,.original}

[hadoop@localhost hadoop]$ cat > /srv/apache-hadoop/etc/hadoop/yarn-site.xml <<"EOF"
<?xml version="1.0" encoding="UTF-8"?>
<?xml-stylesheet type="text/xsl" href="configuration.xsl"?>
<configuration>
	<property>
		<name>yarn.nodemanager.aux-services</name>
		<value>mapreduce_shuffle</value>
	</property>
</configuration>
EOF

单机配置不需要不需要处理slaves保持默认即可

			
[hadoop@localhost hadoop]$ cat /srv/apache-hadoop/etc/hadoop/slaves 
localhost

至此 Hadoop 配置已经完成
